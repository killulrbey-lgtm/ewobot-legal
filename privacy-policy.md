# Privacy Policy

Last Updated: 2026

This Discord bot ("EwoBot") is designed to enhance user experience within Discord servers.

## Data Collection
We may collect limited data such as:
- User ID
- Username
- Server ID
- XP, level, and in-bot currency

We do NOT collect:
- Passwords
- Emails
- IP addresses
- Sensitive personal data

## Data Usage
Data is used only for:
- Level and XP systems
- Economy features
- Profile display
- Bot functionality

## Message Content
Message content is only processed temporarily for:
- Command execution
- Moderation systems
- Activity tracking

We do NOT store message content.

## Data Sharing
We do not sell, share, or distribute user data to third parties.

## User Control
Users can stop using the bot at any time.

## Contact
For any questions, contact the bot developer.
